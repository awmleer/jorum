## 特性

🔩 轻量：简单易用，低侵入性

🔎 清晰：基于RxJS的响应式数据流

⛓ 强类型：使用TypeScript编写

💉 依赖注入：基于修饰器的依赖注入使代码更易于维护

🎣 Hooks支持：为新版React的Hooks而生，但同时也兼容传统语法
